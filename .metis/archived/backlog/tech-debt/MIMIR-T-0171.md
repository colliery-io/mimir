### Technical Debt Impact **[CONDITIONAL: Tech Debt]**
- **Current Problems**: 50+ command handlers in `crates/mimir-dm/src/commands/` duplicate: connection acquisition, error logging, error message formatting. Pattern: `state.db.get_connection().map_err(|e| { error!(...); format!(...) })?` repeated everywhere.
- **Benefits of Fixing**: Consistent error handling and logging, easier to add tracing/metrics, reduced boilerplate per command handler.
- **Risk Assessment**: Low risk - inconsistent error messages across commands, missed logging in some handlers, harder to add cross-cutting concerns.
