# Create Typst templates for campaign documents

*This template includes sections for various types of tasks. Delete sections that don't apply to your specific use case.*

## Parent Initiative

[[MIMIR-I-0017]] - Campaign Document Export

## Objective

Create Typst templates for rendering campaign documents as styled PDFs. Includes a single-document template and a combined template for full campaign export with table of contents.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] Create `campaign/document.typ` - Single document template with:
  - Header showing title, document type, campaign name
  - Rendered Typst content area
  - Footer with "Generated by Mimir"
  - Uses shared styles from `_shared/styles.typ`
- [ ] Create `campaign/combined.typ` - Combined PDF template with:
  - Cover page with campaign name and export date
  - Table of contents listing all documents by level (Campaign/Module/Session/Handout)
  - Page breaks between documents
  - Running headers/footers
- [ ] Templates compile without errors
- [ ] Styling consistent with existing character sheet PDFs

## Implementation Notes

### Files to Create
- `crates/mimir-dm-print/templates/campaign/document.typ`
- `crates/mimir-dm-print/templates/campaign/combined.typ`

### Template Data Structure

**Single Document (`document.typ`):**
```json
{
  "title": "Session 1 Outline",
  "document_type": "session_outline",
  "campaign_name": "Curse of Strahd",
  "content": "= Section Header\n\nConverted Typst content..."
}
```

**Combined PDF (`combined.typ`):**
```json
{
  "campaign_name": "Curse of Strahd",
  "export_date": "2024-01-15",
  "documents": [
    { "title": "...", "document_type": "...", "level": "campaign", "content": "..." },
    { "title": "...", "document_type": "...", "level": "module", "content": "..." }
  ]
}
```

### Dependencies
- MIMIR-T-0141 (markdown converter provides the content format)

## Template Design Specifications

### Single Document Template (`document.typ`)

```
+----------------------------------------------------------+
|  [Campaign Name]                          [Document Type] |
|  ========================================================|
|                                                          |
|  DOCUMENT TITLE                                          |
|  --------------------------------------------------------|
|                                                          |
|  [Converted markdown content rendered here]              |
|                                                          |
|  = Section Header                                        |
|  Body text with *bold* and _italic_ formatting...        |
|                                                          |
|  - Bullet point 1                                        |
|  - Bullet point 2                                        |
|                                                          |
|  | Table | Header |                                      |
|  |-------|--------|                                      |
|  | Cell  | Cell   |                                      |
|                                                          |
+----------------------------------------------------------+
|  Generated by Mimir                          Page X of Y |
+----------------------------------------------------------+
```

**Design Notes:**
- Use `mimir-doc` wrapper with 0.6in margins
- Header: Campaign name left, document type badge right (e.g., "Session Outline")
- Title: Large bold text with accent underline
- Content: Standard body text, preserve markdown structure
- Footer: Centered "Generated by Mimir", page numbers right

### Combined Campaign PDF (`combined.typ`)

**Page 1: Cover Page**
```
+----------------------------------------------------------+
|                                                          |
|                                                          |

```

```
+----------------------------------------------------------+

|  CAMPAIGN DOCUMENTS                                      |
|    Campaign Bible ................................. 3     |

|  MODULE DOCUMENTS                                        |
|    Module 1: The Beginning ....................... 15     |
|    Module 2: Rising Action ....................... 22     |
|                                                          |
|  SESSION DOCUMENTS                                       |
|    Session 1 Outline ............................. 28     |
|    Session 2 Outline ............................. 31     |
|                                                          |
|  HANDOUTS                                                |
|    Player Guidelines ............................. 34     |
|    NPC Quick Reference ........................... 36     |
|                                                          |
+----------------------------------------------------------+
```

**Page 3+: Document Pages**
```
+----------------------------------------------------------+
|  [Campaign Name]    [Document Type]    [Section: Level]  |
|  ========================================================|
|                                                          |
|  DOCUMENT TITLE                                          |
|  --------------------------------------------------------|
|                                                          |
|  [Full document content...]                              |
|                                                          |
+----------------------------------------------------------+
|  Generated by Mimir                          Page X of Y |
+----------------------------------------------------------+

                    --- PAGE BREAK ---

+----------------------------------------------------------+
|  [Next document...]                                      |
```

**Design Notes:**
- Cover page: Centered layout, campaign name prominent
- TOC: Four sections (Campaign/Module/Session/Handout), dot leaders to page numbers
- Documents: Same layout as single template, with running header showing section
- Page breaks between each document
- Continuous page numbering throughout
